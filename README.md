# Marketing Analysis Dashboard

This project demonstrates an end-to-end **marketing analytics pipeline** using **SQL Server, Python, and Power BI** to convert raw marketing and customer data into structured, analytics-ready datasets.

## Overview
The Power BI dashboard provides a consolidated view of marketing performance across products and time periods. It enables stakeholders to analyze conversion efficiency, customer engagement, and review trends through interactive visuals and filters.

## Dashboard Features
- Conversion analysis by product and time period
- Social media engagement tracking including views, clicks, and likes
- Customer review analysis with trend comparisons
- Interactive filters for year, month, and product-level exploration

## Data Pipeline
- **SQL Server:** Data cleaning, transformation, standardization, and de-duplication
- **Python:** Processing customer review and engagement data into structured formats
- **Power BI:** Interactive dashboards for business reporting and monitoring

## Tech Stack
SQL Server | Python | Power BI

## Outcome
The solution supports data-driven marketing decisions by highlighting performance trends, customer behavior patterns, and areas for improvement.
